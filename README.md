تحديث v9 - إصلاح رفع المستهدفين في الحملات

الملفات المعدلة:
- assets/app.js
- campaigns.html

ما تم إصلاحه:
1) قبول عمود الجوال باسم "الجوال" ومرادفاته مثل رقم الجوال، جوال، الهاتف، phone, mobile.
2) دعم CSV المفصول بعلامة Tab أو الفاصلة أو الفاصلة المنقوطة.
3) تنظيف أسماء الأعمدة من BOM والمسافات الخفية والشرطات.
4) تصحيح شرط التحقق من الرقم في صفحة المستهدفين ليعتمد على status === صحيح.
5) جعل إعادة احتساب ملفات المتبرعين بعد رفع المستهدفين اختيارية ومغلقة افتراضياً لتجنب timeout.

طريقة التطبيق:
ارفع الملفات مكان القديمة ثم اعمل Ctrl+F5.
