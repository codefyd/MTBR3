# ولاء SaaS 3.0

## أضيف

- جدول الجمعيات والاشتراكات وحالة التفعيل.
- عضويات المستخدمين والأدوار وصلاحيات ظهور الصفحات.
- لوحة مدير منصة لإنشاء الجمعية والمستخدم الأول وإدارة الاشتراك.
- إنشاء مستخدمين إضافيين وتعديل صفحاتهم وإيقافهم.
- تغيير المستخدم لكلمة مروره المؤقتة من داخل النظام.
- مفاتيح MCP مستقلة للجمعية أو المستخدم، مع إلغاء وانتهاء وسجل تدقيق.
- أدوات MCP للوحة التحكم والمتبرعين والعمليات والحملات والمشاريع.

## تغيّر

- أصبحت المفاتيح الفريدة للعمليات والمتبرعين والمستهدفات والأهداف والتكاليف مرتبطة بالجمعية.
- استُبدلت سياسات `authenticated using (true)` بسياسات عزل حسب العضوية والاشتراك.
- أصبحت دوال التقارير والرفع خاضعة لـRLS بدلاً من تجاوزها.
- أصبحت Caches العمليات والحملات تحمل `organization_id`.

## لم يتغيّر

- صفحات التحليل والتقارير والرفع الحالية ووظائفها الأساسية.
- البيانات القديمة؛ تُنسب إلى جهة افتراضية واحدة أثناء الترقية.
- مفتاح الواجهة يظل المفتاح العام فقط، دون `service_role`.
